# Verkefni 4

Verkefnið er byggt á verkefni 2 og er útfærsla á klassískum lúdó leik.

Forritið er byggt með Java 25, JavaFX 25.0.1 og verkefnisstjórnunartólinu Maven 4.0.0 (sem er samhæft með Maven 3).

## Fyrir keyrslu
Gangið úr skugga að þið eruð með fyrrnefnd forritunarsöfn
- Java 25.0.0
- JavaFX 25.0.1
- Maven 4.0.0 (sem er samhæft með Maven 3)

Einnig að þið eruð með JDK 23 eða nýrra, og IntelliJ IDEA eða annað þróunarumhverfi sem keyrir JavaFX með Maven.
Passið að Maven og IntelliJ noti sama JDK.
mvn -v

## Keyrsla
Sækið jar skrá úr nýjustu útfærslu verkefnisins, opnið skránna í IntelliJ. Ýtið á m hægra megin á skjánum. Þar undir er ýtt á javafx og síðan er tvísmellt á javafx:run til að keyra forritið. 
Ef þú villt keyra forritið á skipanalínu er hægt að nota "mvn clean javafx:run", passa þarf að vera með Maven sett upp á vélinni þinni.

Ef villa kemur upp prófið að eyða target möppunni. Ýtið þar næst á takka, sem staðsettur er efst innan glugga mavens (m), sem heitir "Generate Sources and Update Folders For All Projects" sem lýtur út eins og mappa með bláu update hjóli ofan á.
